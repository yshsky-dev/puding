# 🐹 푸딩이를 기억하며

푸딩이 추모 사이트입니다.

## 배포 방법 (GitHub Pages)

1. GitHub에서 새 repository 만들기
2. 이 폴더 전체를 push
3. repository Settings → Pages → Source: `main` branch → Save
4. 잠시 후 `https://[username].github.io/[repo-name]` 으로 접속 가능
